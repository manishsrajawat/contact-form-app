API Overview

This API provides endpoints for managing contact form submissions. It supports creating new contacts and retrieving existing ones from a MongoDB database.

Base URL

http://localhost:3000

Content Type

All requests and responses use application/json content type.

Note: The API supports both live database mode and demo mode. In demo mode, contact submissions are logged but not actually saved to the database.
POST

Create Contact

POST /api/contact

Creates a new contact entry in the database.

Request Body

Field Type Required Description
name string Yes Contact's full name (trimmed)
email string Yes Contact's email address (lowercase, trimmed)
phone string Yes Contact's phone number (trimmed)

Response Codes

Code Description
201 Contact created successfully
400 Missing required fields or validation error
500 Server error

Success Response Example

{ "success": true, "message": "Your details have been saved successfully!", "contact": { "name": "John Doe", "email": "john.doe@example.com", "phone": "+1234567890", "createdAt": "2025-08-13T08:47:48.014Z" } }

Error Response Example

{ "success": false, "message": "Please provide all required fields: name, email, and phone" }
GET

Get All Contacts

GET /api/contacts

Retrieves all contacts from the database, sorted by creation date (newest first).

Response Codes

Code Description
200 Contacts retrieved successfully
500 Server error

Success Response Example

{ "success": true, "count": 3, "contacts": [ { "_id": "689c5134d5ce554014b86346", "name": "1.localmongo", "email": "local@mongo.db", "phone": "999999999", "createdAt": "2025-08-13T08:47:48.014Z", "__v": 0 }, { "_id": "689aefe2bbcc212abfe5414a", "name": "222", "email": "222@222.com", "phone": "2222222", "createdAt": "2025-08-12T07:40:18.161Z", "__v": 0 } ] }

Data Models

Contact Model

Field Type Description
_id ObjectId Unique MongoDB identifier
name String Contact's full name (required, trimmed)
email String Contact's email (required, lowercase, trimmed)
phone String Contact's phone number (required, trimmed)
createdAt Date Timestamp when contact was created
__v Number Mongoose version key

Usage Examples

JavaScript/Fetch API

// Create a new contact const createContact = async () => { try { const response = await fetch('http://localhost:3000/api/contact', { method: 'POST', headers: { 'Content-Type': 'application/json', }, body: JSON.stringify({ name: 'Jane Smith', email: 'jane.smith@example.com', phone: '+1-555-0123' }) }); const result = await response.json(); console.log(result); } catch (error) { console.error('Error:', error); } }; // Get all contacts const getAllContacts = async () => { try { const response = await fetch('http://localhost:3000/api/contacts'); const result = await response.json(); console.log(`Found ${result.count} contacts:`, result.contacts); } catch (error) { console.error('Error:', error); } };

cURL Examples

# Create a new contact curl -X POST http://localhost:3000/api/contact \ -H "Content-Type: application/json" \ -d '{ "name": "Mike Johnson", "email": "mike.johnson@example.com", "phone": "+1-555-9876" }' # Get all contacts curl -X GET http://localhost:3000/api/contacts
